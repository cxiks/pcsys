🔧 Bash Scripts Collection
Набор полезных bash-скриптов для повседневных задач разработчика.

📋 Содержимое
Скрипт	Описание
greeting.sh	Приветствие пользователя по имени
sum_calculator.sh	Сложение двух чисел
even_odd_checker.sh	Проверка четности числа
project_structure_creator.sh	Создание структуры веб-проекта
line_counter.sh	Подсчет строк в файле
password_generator.sh	Генератор паролей
file_finder.sh	Поиск файлов по расширению
🚀 Быстрый старт
bash
# Клонировать репозиторий
git clone https://github.com/ваш-username/bash-scripts.git
cd bash-scripts

# Дать права на выполнение
chmod +x *.sh

# Запустить любой скрипт
./greeting.sh
